# Agent Skills for SOMA

Agent skills that teach AI assistants how to participate in [SOMA](https://soma.org) — a network that trains a foundation model through competition.

## Available Skills

### [`soma`](./soma/)

Expert guide and competitive playbook for SOMA network participation. Covers data submission, model training, reward claiming, and strategic optimization. Endows any agent with deep knowledge of SOMA's architecture, SDK, CLI, and the strategic thinking required to compete effectively and contribute to the network's collective intelligence.

**Use cases:**
- Submit data to earn SOMA rewards
- Train and publish models via commit-reveal
- Claim rewards and manage staking
- Optimize strategies for competitive advantage
- Set up development environments (localnet, testnet)

## Installation

```bash
npx skills add soma-org/skills
```

### Manual

Clone and place in your agent's skills directory:

```bash
git clone https://github.com/soma-org/skills.git
# Copy soma/ folder to your agent's skill directory
```
